# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Semantic Versioning](https://semver.org/lang/fr/).

## [0.2.0] - 2025-12-28

### Ajouté
- Système de thèmes complet avec configurations étendues
- Nouveaux composants UI :
  - LoadingSkeleton pour les états de chargement modernes
  - Modal avec design glassmorphism
  - SearchBar avec animations améliorées
  - Stat pour l'affichage de métriques
  - Toggle pour la fonctionnalité de switch
- Internationalisation (i18n) :
  - Support Anglais
  - Support Français
- ThemeContext pour la gestion centralisée des thèmes
- Exports centralisés des composants UI

### Modifié
- Refonte complète de GameDetails avec meilleure UX
- Amélioration de l'interface des pages Home, Settings, Games
- Mise à jour de tous les modaux avec le nouveau design
- Configuration Tailwind étendue avec gradients et ombres

### Technique
- Auto-update désactivé temporairement
- Configuration pour distribution manuelle
